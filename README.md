# Hopscotch CLI

An opinionated approach to modular JavaScript.

## Installation

### npm

```bash
npm install -g hopscotch-cli
```

## Usage

```bash
hopscotch init <directory>
```

> NOTE: If no `directory` argument is given, the project will be installed in the current deirctory.
